# Just Ship Things

A showcase of products shipped by [Vishnu](https://heyvish.com)

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Shadcn UI
- Framer Motion

## Features

- Dark/Light mode
- Responsive design
- Beautiful animations
- Product showcase
- SEO optimized
